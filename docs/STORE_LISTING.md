## 전체 설명 (복붙용)

```
Drop은 DGIST 배드민턴 동아리를 위한 코트 예약·매칭·출석·포인트·레슨 통합 앱입니다.

■ 주요 기능
• S1 체육관 9개 코트를 실제 배치로 한눈에 확인
• 빈 코트 예약, 합류 신청, 난타/경기 모드, 점수 기록
• 활동 시간(화·목 18:30–21:50)에 맞춰 현장 중심으로 운영
• 체육관 근처(약 500m)에서만 예약·출석·봉사 인증 (위치 권한)
• 출석·청소·네트 봉사로 포인트 적립, 코트 이용 시 포인트 사용
• 친구·일정, 파트너 모집방, 코치 레슨 대기열
• 관리자 승인·운영 도구 (운영진 계정)

■ 이런 분께 추천
동아리 활동일에 코트 잡기·누가 왔는지·레슨 순서까지 한 앱으로 보고 싶을 때.

■ 안내
• 본 앱은 동아리 내부 운영용입니다.
• 일부 기능은 회원 등급·관리자 권한·활동 시간에 따라 제한됩니다.
• 원격 푸시 알림은 기기에서 알림을 허용한 경우에만 전송됩니다.

문의: 동아리 Drop 운영진
개인정보처리방침: (배포 URL)/privacy
```
