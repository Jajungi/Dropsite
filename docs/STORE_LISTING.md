## 짧은 설명

DGIST Drop 동아리 코트 예약·출석·매칭·포인트.

---

## 전체 설명

```
Drop은 DGIST 배드민턴 동아리를 위한 코트 예약·매칭·출석·포인트·레슨 통합 앱입니다.

■ 주요 기능
• S1 체육관 9개 코트를 실제 배치로 확인
• 빈 코트 예약, 합류 신청, 난타/경기 모드, 점수 기록
• 활동 시간(화·목 18:30–21:50) 기준 현장 운영
• 체육관 근처(약 500m)에서만 예약·출석·봉사 인증
• 출석·청소·네트 봉사 포인트 적립, 코트 이용 시 포인트 사용
• 친구·일정, 파트너 모집방, 코치 레슨 대기열
• 운영진 계정용 관리 기능

■ 안내
• 본 앱은 동아리 내부 운영용입니다.
• 일부 기능은 회원 등급·관리자 권한·활동 시간에 따라 제한됩니다.
• 원격 푸시 알림은 기기에서 알림을 허용한 경우에만 전송됩니다.

문의: 동아리 Drop 운영진
개인정보처리방침: https://<배포-도메인>/privacy
```
